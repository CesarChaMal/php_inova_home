Why does the calendar appear to be "squished" in some cases?

This can happen if the calendar is created as part of a block element that has a specified width small enough to cause the calendar to contract.  This can be fixed by moving the calendar outside of this block, or specifying a wider area. Table cells are notorious for this.

My calendar does not get added to the page--like it's not even created!

Most likely there is an error in your page DOM.  DateClick is a DHTML calendar, and if the HTML syntax on your page is not valid, it may not appear correctly.  Please run an HTML validator such as CSE HTML Validator available from www.htmlvalidator.com. In general make sure your HTML tags are properly "nested" and you have a closing tag for every opening tag.
